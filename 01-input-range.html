This is a test case designed to determine how the <input type="range"> element is announced by various screen readers.

Example

Code

<div>
  <label for="range">Choose a range</label>
  <input type="range" id="range">
</div>

Assistive technologies

VoiceOver

  • Chrome: 50. Choose a range. Slider.
  • Firefox: 50. Choose a range. Slider.
  • Safari: 50. Choose a range. Slider.

NVDA

  • Chrome: Choose a range. Slider. 50.
  • Firefox: Choose a range. Slider. 50.
  • Edge: Choose a range. Slider. 50.

JAWS

  • Chrome: Choose a range. Left Right slider. 50.
  • Firefox: Choose a range. Left right slider. 50.
  • Edge: Choose a range. Left right slider. 50.
