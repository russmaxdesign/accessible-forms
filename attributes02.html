Example

This is a test case designed to determine how the aria-valuemax attribute is announced. In real situations, authors should always provide clear instructions on how to successfully complete form questions.

Code

<div class="form-group">
  <label for="number">Choose a number</label>
  <input 
    type="number"
    id="number"
    aria-valuemax="20"
  >
</div>

Assistive technologies

  • Firefox: Choose a number. Stepper. [aria-valuemax attribute not announced]
  • Safari: Choose a number. Incrementable. Edit text. [aria-valuemax attribute not announced]

NVDA

  • Chrome: Choose a number. Spin button. Editable. Blank. [aria-valuemax attribute not announced]
  • Firefox: Choose a number. Spin button. Has autocomplete. Editable. Blank. [aria-valuemax attribute not announced]
  • Edge: Choose a number. Spin button. Editable. Blank. [aria-valuemax attribute not announced]

JAWS

  • Chrome: Choose a number. Edit. Spin box. To set the value use the arrow keys or type the value. [aria-valuemax attribute not announced]
  • Firefox: Choose a number. Edit. Spin box. To set the value use the arrow keys or type the value. [aria-valuemax attribute not announced]
