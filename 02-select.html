This is a test case designed to determine how the <select> element is announced by various screen readers.

Example

Code

<div>
  <label for="suburb">Suburb</label>
  <select id="suburb">
    <option value="Abbotsbury">Abbotsbury</option>
  </select>
</div>

Assistive technologies

VoiceOver

  • Chrome: Abbotsbury. Suburb. Menu popup collapsed. Button.
  • Firefox: Abbotsbury. Suburb. Popup button.
  • Safari: Abbotsbury. Suburb. Popup button.

NVDA

  • Chrome: Suburb combobox. Abbotsbury. Collapsed.
  • Firefox: Suburb combobox. Abbotsbury. Collapsed.
  • Edge: Suburb combobox. Abbotsbury. Collapsed.

JAWS

  • Chrome: Suburb combobox. Abbotsbury.
  • Firefox: Suburb combobox. Abbotsbury.
  • Edge: Suburb combobox. Abbotsbury.
