This is a test case designed to determine how the <input type="email"> element is announced by various screen readers.

Example

Code

<div>
  <label for="email">Email address</label>
  <input type="email" id="email">
</div>

Assistive technologies

VoiceOver

  • Chrome: Email address. Edit text.
  • Firefox: Email address. Edit text.
  • Safari: Email address. Edit text.

NVDA

  • Chrome: Email address. Edit blank.
  • Firefox: Email address. Edit. Has autocomplete. Blank.
  • Edge: Email address. Edit. Blank.

JAWS

  • Chrome: Email address. Edit. Type in text.
  • Firefox: Email address. Edit. Type in text.
  • Edge: Email address. Edit. Type in text.
